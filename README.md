## Run Locally

Clone the project

```bash
  git clone https://github.com/DevShaunB/fem-social-proof-section.git
```

Go to the project directory

```bash
  cd fem-social-proof-section
```

Run `index.html`

```bash
  <browsername> index.html
```

E.g.

```bash
  firefox index.html
```

```bash
  google-chrome index.html
```
